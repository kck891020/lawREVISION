🌐 웹페이지 이식 가이드

아래 코드를 그대로 복사해서 원하시는 웹페이지에 붙여넣으시면 됩니다.
HTML 파일로 저장하거나, 워드프레스 등 CMS의 HTML 블록에 삽입하세요.

① 검색 위젯 삽입 코드
<!-- 소방법령 검색 위젯 --> <div id="fireLawWidget"></div> <script> const API_KEY = "kck891020"; const BASE = "https://www.law.go.kr/DRF"; async function searchFireLaw(query) { const url = `${BASE}/lawSearch.do?OC=${API_KEY}&target=law&type=JSON&query=${encodeURIComponent(query)}`; const res = await fetch(url); const data = await res.json(); return data.LawSearch?.law || []; } // 검색 후 결과를 HTML로 렌더링 searchFireLaw("소방기본법").then(laws => { const widget = document.getElementById("fireLawWidget"); widget.innerHTML = laws.map(l => `<div class="law-item"> <h3>${l.법령명한글}</h3> <p>공포일: ${l.공포일자} | 시행일: ${l.시행일자}</p> </div>` ).join(""); }); </script>
② 현재 추출된 데이터를 JSON으로 내보내기
추출된 데이터가 없을 경우 먼저 검색 또는 조문 조회를 실행해주세요.
